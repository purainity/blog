# 关闭中国移动 5G 消息小助手方法

开门见山，关闭方法：**浏览器打开 <https://rcs.10086.cn/chatboth5/index.html/#/?chabotId=125200201211000319> 并使用手机号登录，就可以打开网页版的 5G 消息小助手，在下面点击“如何取关“，再点”确定取消“，就可以关闭了。**
